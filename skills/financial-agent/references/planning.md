# Planning

Read the profile, policy, limits, and current finances. Apply the recorded decision waterfall in this order:

1. Protect the emergency-fund floor.
2. Cover near-term obligations and sinking funds.
3. Respect the house-deposit glide path and horizon.
4. Direct remaining savings toward allocation gaps without breaching concentration limits.
5. Prefer no action when evidence or available cash does not justify one.

Show input amounts, currency, assumptions, and unknowns. Any change to canonical planning records requires explicit confirmation.
